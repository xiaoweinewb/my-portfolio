# 星空摄影指南

拍摄星空需要耐心、时机和一点点运气。这是我多次拍摄后总结的一些经验和技巧，希望对你有帮助。

## 准备工作

### 选择合适的地点

- **远离光污染** — 城市灯光会影响星空的可见度
- **天气晴朗** — 云层会遮挡星空
- **注意月相** — 新月前后最佳，满月时月光太亮

### 必备器材

1. **相机** — 最好选择全画幅相机，高感光度表现更好
2. **广角镜头** — 14-24mm 是拍摄银河的黄金焦段
3. **三脚架** — 稳定的支撑是长曝光的基础
4. **快门线** — 避免按快门时造成相机抖动

## 拍摄参数建议

### 银河拍摄

```
光圈: f/2.8 或更大
快门: 15-25秒（避免星轨）
ISO: 1600-6400
对焦: 手动对焦到无穷远
```

### 流星雨拍摄

- 使用更短的快门时间（8-15秒）
- 连续拍摄多个小时
- 多机位增加捕捉几率

## 后期处理

适当的后期可以提升照片的观感：

1. **调整色温** — 让星空呈现更自然的蓝调
2. **提升清晰度** — 让星星更加锐利
3. **降噪处理** — 高ISO照片通常需要降噪
4. **裁剪构图** — 突出画面中最美的部分

## 注意事项

> 安全第一！夜间拍摄要注意人身安全，最好结伴同行。
